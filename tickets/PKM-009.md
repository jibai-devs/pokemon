## Summary

Build `scripts/train_bc.py` — train a policy network on top-agent replay data via supervised imitation learning.

## Network architecture

```
board_state (434d)
    │
BoardEncoder (2-layer MLP, 256d hidden)
    │
board_vec (256d)
    │                           ┌── value head: MLP → scalar
    │                           │   (for RL fine-tuning later)
    │  for each option_i (64d): │
    │    option_vec_i           │
    │    score_i = MLP(cat(board_vec, option_vec_i)) → logit
    │
policy = softmax(scores over available options)
```

## Training

- Loss: cross-entropy on chosen option index (or multi-label for maxCount > 1)
- Optimizer: Adam, lr=1e-3 with cosine decay
- Batch: variable-length option lists — pad to max options in batch, mask padding in softmax
- Validation: held-out 10% of replays (not trained on)
- Checkpoint: save best val loss

## Success criterion

Trained policy win-rate vs random agent > 70% over 100 games (run in WSL via `python -m pokemon`).

## Notes

- Start simple: MLP encoder only. No transformer/attention until MLP is validated.
- Board state and option vectors are pre-computed by featurize.py (PKM-008) — training loop loads .npz, not raw replays
- Value head output is unused in BC phase; initialise it but don't train it
- Save card/attack embedding weights with checkpoint — needed for inference in agent.py
