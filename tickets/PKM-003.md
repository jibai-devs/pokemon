## Summary

Package `main.py` and `deck.csv` into a `.tar.gz` and upload it to the Kaggle competition page.

## Steps

1. From the repo root: `tar -czvf submission.tar.gz main.py deck.csv`
2. Upload under the "My Submissions" tab on the Simulation competition page
3. Wait for the validation game (agent vs itself) to complete
4. Confirm status is not `Error` — download agent logs if it fails

## Acceptance criteria

- Submission passes validation (no `Error` status)
- Agent enters the matchmaking pool

## Notes

- Submission size limit: 197.7 MiB — not a concern for this bundle
- The platform runs one game of the agent vs itself to validate before matchmaking
- If validation fails, logs are downloadable from the Submissions tab
- Only `main.py` and `deck.csv` are needed for the minimal submission — no other files required
