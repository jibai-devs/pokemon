## Summary

The competition requires `main.py` at the top level of the `.tar.gz`. It must expose an `agent(obs_dict) -> list[int]` function. The file cannot import from `src/pokemon/` because the competition container only has the files in the bundle — everything lands flat at `/kaggle_simulations/agent/`.

For now the agent just needs to make **legal moves**, not optimal ones. The engine only ever presents legal options, so picking randomly from the presented options is always valid.

## Acceptance criteria

- `main.py` exists at the repo root
- Exposes `agent(obs_dict: dict) -> list[int]`
- Reads deck from `deck.csv` at import time (relative to `__file__`)
- Returns the deck when `obs_dict["select"] is None` (local testing compat)
- Otherwise returns `maxCount` random indices from the available options
- No imports from `src/pokemon/` — fully self-contained
- Runs locally: `uv run python -c "import main; print('ok')"` should pass

## Notes

- The `select` dict has `option` (list of legal choices) and `maxCount` (how many to pick)
- `random.sample(range(len(options)), maxCount)` is sufficient for a legal random agent
- The platform runs a validation game (agent vs itself) before entering matchmaking — this must pass
